# Superstore Sales Data Analysis

This project focuses on performing **Exploratory Data Analysis (EDA)** on a retail sales dataset to understand business performance, identify profitability drivers, and uncover insights that can support data-driven decision making.

The analysis explores relationships between **sales, profit, discounts, customer segments, product categories, and regions** to understand how different factors affect business outcomes.

---

## Project Objectives

- Clean and preprocess the retail sales dataset
- Perform exploratory data analysis (EDA)
- Identify patterns in sales and profitability
- Analyze the impact of discounts on profit
- Understand customer segments and regional performance
- Generate business insights from the data

---

## Dataset Overview

The dataset contains sales transactions from a retail superstore.  
Each row represents an order with information about:

- Order ID
- Order Date
- Ship Date
- Customer Segment
- Region
- Category and Sub-Category
- Sales
- Quantity
- Discount
- Profit

This dataset is commonly used for learning **business analytics and exploratory data analysis**.

## Key Analysis Performed

### Data Cleaning
- Checked for missing values
- Verified data types
- Handled inconsistent or unnecessary columns

### Univariate Analysis
- Distribution of sales
- Distribution of profit
- Category and sub-category frequency analysis

### Bivariate Analysis
- Relationship between **Discount and Profit**
- Sales comparison across **regions and categories**
- Profit comparison across **customer segments**

### Correlation Analysis
- Identified relationships between numerical variables
- Used **correlation heatmap** to visualize patterns

---

## Key Business Insights

Some important insights from the analysis include:

- Higher discounts often lead to **lower profitability**
- Certain product categories generate **high revenue but lower profit margins**
- Regional sales performance varies significantly
- Some sub-categories contribute strongly to total sales
- Understanding customer segments can help optimize marketing strategies
